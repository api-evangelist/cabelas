# Cabela's (cabelas)
Cabela's is a specialty retailer of hunting, fishing, camping, and outdoor recreation merchandise, operating retail stores and an e-commerce platform. Cabela's is a subsidiary of Bass Pro Shops following the 2017 acquisition, and continues to operate under its own brand with retail locations and a consumer-facing web storefront.
